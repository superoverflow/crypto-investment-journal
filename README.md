## getting started
- make `.env.local`
```
GOOGLE_CLIENT_EMAIL=<service-account-name>@<your gcp project>.iam.gserviceaccount.com
GOOGLE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n......\n-----END PRIVATE KEY-----\n"
GOOGLE_SHEET_ID=<google sheet id>
```
- run 
```
npm run dev
```
